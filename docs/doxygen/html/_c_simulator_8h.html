Detailed Description

Defines the functions and structures for simulating and evaluating circuits in a circuit modeling framework.

The CSimulator module provides functions for evaluating the performance, grade, and recovery of circuits in a circuit modeling framework. It includes definitions for circuit parameters and multiple overloads for evaluating circuits with or without additional parameters.
