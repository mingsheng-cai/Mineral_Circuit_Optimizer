\doxysubsection{Detailed Description}
Abstract base class for genetic algorithm operators. 

The \doxylink{class_operator}{Operator} class provides an abstract interface for genetic algorithm operators. It defines a pure virtual function {\ttfamily \doxylink{class_operator_acbd2e97e189d862aa139b3c334a18619}{apply()}}, which must be implemented by derived classes. This class serves as a base for various types of genetic algorithm operators, such as selection, crossover, and mutation.
