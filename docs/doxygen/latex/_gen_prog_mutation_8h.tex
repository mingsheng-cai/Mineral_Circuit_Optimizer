\doxysubsection{Detailed Description}
Defines the \doxylink{class_gen_prog_mutation}{Gen\+Prog\+Mutation} class for performing mutation operations in a genetic programming algorithm. 

The \doxylink{class_gen_prog_mutation}{Gen\+Prog\+Mutation} class provides methods to perform mutation operations on individuals in a genetic programming algorithm. This class is designed to introduce variations into the population by modifying individuals\textquotesingle{} genetic material, which can then be used in subsequent generations of the algorithm.
