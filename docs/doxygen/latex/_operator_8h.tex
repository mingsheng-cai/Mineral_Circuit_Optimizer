\doxysubsection{Detailed Description}
Defines the abstract base class for genetic algorithm operators. 

The \doxylink{class_operator}{Operator} class serves as an abstract base class for various genetic algorithm operators, such as selection, crossover, and mutation. Each specific operator must inherit from this class and implement the pure virtual function {\ttfamily apply()}. This design allows for a consistent interface and enables polymorphism, making it easier to manage and apply different operators within a genetic algorithm framework.
