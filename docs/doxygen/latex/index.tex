This is an implementation for the {\bfseries{ACS -\/ Gerardium Rush}} project delivered by group {\bfseries{Pentlandite}}.

This project is designed to optimize circuit configurations using genetic algorithms to enhance the recovery of valuable materials in separation technologies. The goal is to create an efficient and effective tool for optimizing circuits in minerals processing, particularly for the recovery of "{}gerardium"{}. This project combines the power of genetic algorithms with mass balance simulations to find the optimal circuit configuration.\hypertarget{index_autotoc_md113}{}\doxysubsection{\texorpdfstring{ğŸ”­ Overview}{ğŸ”­ Overview}}\label{index_autotoc_md113}
The project is organized into several key components, each with specific responsibilities\+:


\begin{DoxyEnumerate}
\item {\bfseries{\doxylink{class_circuit}{Circuit} Simulation (\doxylink{_c_simulator_8cpp}{CSimulator.\+cpp})}}\+:


\begin{DoxyItemize}
\item Core simulation logic for evaluating circuit performance
\item Simulates the mass balance for given circuit configurations.
\item Utilizes classes like {\ttfamily CCircuit} and {\ttfamily \doxylink{class_c_unit}{CUnit}} to represent and manage circuit units and their interactions.
\end{DoxyItemize}
\item {\bfseries{Genetic Algorithm (\doxylink{_genetic___algorithm_8cpp}{Genetic\+\_\+\+Algorithm.\+cpp})}}\+:


\begin{DoxyItemize}
\item Implements the genetic algorithm for optimizing circuit configurations.
\item Includes various operators for selection, crossover, and mutation.
\item Parallel processing enabled using Open\+MP for efficient computation.
\end{DoxyItemize}
\end{DoxyEnumerate}\hypertarget{index_autotoc_md114}{}\doxysubsection{\texorpdfstring{âœ¨ Features}{âœ¨ Features}}\label{index_autotoc_md114}

\begin{DoxyItemize}
\item {\bfseries{Genetic Algorithm Optimization}}\+: Efficiently explore numerous circuit configurations using genetic algorithms to find the most effective setups.
\item {\bfseries{Mass Balance Simulation}}\+: Simulate the mass balance for each circuit to ensure accurate recovery and grade calculations, ensuring the reliability of the proposed solutions.
\item {\bfseries{Parallel Processing}}\+: Leverage Open\+MP for faster computations and simulations, enhancing performance and reducing computation time.
\item {\bfseries{Comprehensive Unit Tests}}\+: Ensure the correctness of the circuit simulator and genetic algorithm through extensive unit tests, providing a robust and reliable tool.
\item {\bfseries{Visualization Tools}}\+: Utilize Python scripts and Jupyter notebooks for post-\/processing and visualizing the results, helping to analyze and interpret the outcomes effectively.
\item {\bfseries{Documentation}}\+: Detailed API documentation and project reports to facilitate understanding and usage, making it easier for users to get started and for developers to extend the project.

{\bfseries{Genetic Algorithm Operators}}


\begin{DoxyItemize}
\item {\bfseries{Selection}}\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{class_nary_tournament_selection}{Nary\+Tournament\+Selection}}\+: Selects the best individuals based on tournament competition.
\item {\ttfamily \doxylink{class_rank_selection}{Rank\+Selection}}\+: Selects individuals based on their rank within the population.
\item {\ttfamily \doxylink{class_roulette_wheel_selection}{Roulette\+Wheel\+Selection}}\+: Selects individuals based on their fitness proportionally.
\end{DoxyItemize}
\item {\bfseries{Crossover}}\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{class_two_point_crossover}{Two\+Point\+Crossover}}\+: Combines genes from two parents at two crossover points.
\item {\ttfamily \doxylink{class_uniform_crossover}{Uniform\+Crossover}}\+: Combines genes from two parents based on a uniform distribution.
\item {\ttfamily \doxylink{class_pure_single_point_crossover}{Pure\+Single\+Point\+Crossover}}\+: Combines genes from two parents at a single crossover point.
\end{DoxyItemize}
\item {\bfseries{Mutation}}\+:
\begin{DoxyItemize}
\item {\ttfamily \doxylink{class_guided_mutation}{Guided\+Mutation}}\+: Performs guided mutation based on specific rules or heuristics.
\item {\ttfamily \doxylink{class_gen_prog_mutation}{Gen\+Prog\+Mutation}}\+: Performs genetic programming mutation by changing operations and parameters.
\end{DoxyItemize}
